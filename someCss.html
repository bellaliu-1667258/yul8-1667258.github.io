Style and Tags

You apply one or more property:value pairs to a tag to "style" that tag. Usually, all the property:value pairs are kept outside the HTML. But to get you started with CSS, we'll show you how to type them right inside the tag you want to style. To add a style to a tag, you simply type inside it and put the property:value pairs inside the quotes.
